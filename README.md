# TFM_MultiAgentes_Test 🧪

**Nota Importante:** Este repositorio es exclusivamente una versión de pruebas (sandbox).

Este proyecto contiene el entorno de experimentación para el ecosistema de agentes de IA autónomos enfocado en la optimización de carteras multiactivo. 

El objetivo principal de este repositorio es servir como espacio seguro para probar nuevas características, refactorizaciones de código y validaciones sin riesgo de afectar el código fuente principal del Trabajo de Fin de Máster (TFM).

## Características de este entorno
* 🛠️ **Pruebas de desarrollo:** Espacio para testear la arquitectura multiagente y los scripts de ejecución.
* ⚠️ **Código experimental:** El estado de los archivos y ramas en este repositorio puede ser inestable o estar en construcción.
* 🔄 **Aislamiento:** Permite la validación de integraciones antes de llevar los cambios a la versión final.
